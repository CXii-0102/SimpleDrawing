##  项目概述

本项目是一个功能完整的参数曲线与曲面绘图系统，在原有绘图系统基础上，手动实现了经典的Bézier曲线、B样条曲线和Bézier曲面算法。所有核心算法均基于数学原理从零实现，未使用任何现成的数值计算库。

##  主要功能

###  参数曲线
- **Bézier曲线**
  -  Bernstein基函数方法
  -  de Casteljau递推算法
  -  支持任意次数（常用2-4次）
  
- **B样条曲线**
  -  Cox-de Boor递推算法
  -  二次/三次B样条
  -  均匀/夹紧节点向量
  -  局部控制特性

###  参数曲面
- **双三次Bézier曲面**
  -  张量积原理实现
  -  4×4控制网格
  -  网格线显示模式
  -  渐变填充模式

###  交互功能
-  控制点拖拽实时更新
-  平移/旋转/缩放变换
-  实时预览绘制过程
-  视图缩放
-  保存/打开/导出

###  基础功能（第二次作业集成）
-  直线、矩形、圆形、多边形
-  颜色和线宽设置
-  图形选择与拖拽

##  快速开始

### 环境要求
- Python 3.x
- PyQt5

### 安装运行
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行程序
python main.py

# 3. 运行测试（可选）
python test_curves.py
```

### 基本操作
1. **绘制Bézier曲线**: 点击"Bézier曲线" → 点击画布添加控制点 → 双击完成
2. **绘制B样条**: 点击"B样条曲线" → 添加4个以上控制点 → 双击完成
3. **创建曲面**: 点击"Bézier曲面" → 拖拽控制点调整形状
4. **编辑图形**: 选择工具 → 点击图形 → 拖拽控制点或应用变换
